Ednova Consulting - Professional Landing Page

This repository contains the official landing page for Ednova Consulting, a premier consulting firm specializing in Intelligent Automation (RPA) and Modern Web Development.
🚀 Tech Stack

    HTML5 / CSS3 (Semantic structure)

    Tailwind CSS (Utility-first styling)

    JavaScript (Lightweight i18n for English/Spanish support)

    Formspree (Serverless contact form handling)

    Calendly API (Automated appointment scheduling and Google Calendar sync)

🛠️ Features

    Bilingual Support: Dynamic language switching between English and Spanish.

    Service Tiers: Clearly defined offerings for Branding, Digital Presence, and RPA.

    Mobile First: Fully responsive design optimized for all devices.

    Direct Conversion: Integrated WhatsApp button and automated scheduling system.
